# ETL com Python (IA Simulada)

## 📌 Descrição
Pipeline ETL que lê dados de um CSV, gera mensagens personalizadas simulando IA e salva o resultado.

## ⚙️ Tecnologias
- Python
- Pandas

## 🔄 Pipeline
1. Extract → lê CSV
2. Transform → gera mensagens
3. Load → salva novo CSV

## ▶️ Como executar

```bash
pip install -r requirements.txt
python src/main.py
