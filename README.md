etl-genai-local
Projeto ETL com Python simulando IA generativa

ETL com Python (IA Simulada)
📌 Descrição
Pipeline ETL que lê dados de um CSV, gera mensagens personalizadas simulando IA e salva o resultado.

⚙️ Tecnologias
Python
Pandas
🔄 Pipeline
Extract → lê CSV
Transform → gera mensagens
Load → salva novo CSV
▶️ Como executar
pip install -r requirements.txt
python src/main.py
